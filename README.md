# Portfolio (Jaden Mack)

This is a starter portfolio inspired by vanxh.dev. It includes:

- A dark layout with profile card
- A live clock showing MDT (America/Denver) updating every second
- A live birthday countdown to Feb 17 updating every second
- Projects folder with hover animation (letter pops out)
- Card hover physics using framer-motion

To run locally:

1. Install dependencies

```bash
npm install
```

2. Run dev server

```bash
npm run dev
```

Notes:
- The email send and "what I use" areas are intentionally omitted per request.
- Framer Motion is already added to `package.json`.
